This folder contains shader programs that generalize across dimensions and geometry types.

## Categories
I use 3-letter abbreviations for the shader stages here:
- `set`: `setup`
- `bgn`: `begin`
- `sdw`: `shadow`
- `sdc`: `shadowcomp`
- `pre`: `prepare`
- `geo`: `gbuffers_*`
- `dfr`: `deferred`
- `com`: `composite`
